---
title: "List Indexes"
mode: "wide"
---

Returns a list of all encrypted index names accessible via the client at the set `DBConfig`.

```cpp
std::vector<std::string> ListIndexes();
```

### Returns

`std::vector<std::string>`: A list of index names.

### Exceptions

<AccordionGroup>
    <Accordion title="std::runtime_exception">
        - Throws if the list of indexes could not be retrieved.
    </Accordion>
</AccordionGroup>

### Example Usage

```cpp
#include "cyborgdb/client.hpp"

// ... Initialize the client ...

auto indexes = client.ListIndexes();

// Print the list of indexes
for (const auto& name : indexes) {
    std::cout << name << std::endl;
}
```
