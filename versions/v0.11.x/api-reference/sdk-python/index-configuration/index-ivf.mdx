Creates a configuration for an Inverted File (IVF) index with vector quantization. IVF indexes provide a balance between search accuracy and performance by clustering vectors and using quantization to reduce memory usage.

| Speed | Accuracy | Memory Usage |
|-------|----------|--------------|
| Fast  | Good     | Medium       |

```python
IndexIVF(
    dimension,
    n_lists,
    metric="euclidean"
)
```

### Parameters
| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `dimension` | `int` | - | The dimensionality of the vectors to be indexed |
| `n_lists` | `int` | - | Number of clusters (centroids) for the inverted file structure |
| `metric` | `str` | `"euclidean"` | Distance metric for similarity calculations |

Supported metrics:
- `"euclidean"`: Euclidean distance
- `"cosine"`: Cosine similarity
- `"l2"`: L2 distance

### Returns

An `IndexIVF` configuration object that can be passed to `client.create_index()`.

### Example Usage

```python
from cyborgdb import IndexIVF

# Create a standard IVF configuration
config = IndexIVF(
    dimension=384,
    n_lists=100,
    metric="cosine"
)

# Create an IVF index with the specified configuration
index = client.create_index(
    index_name=index_name,
    index_key=index_key,
    index_config=config
)
```
