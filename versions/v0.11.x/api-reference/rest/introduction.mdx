The CyborgDB REST API provides HTTP endpoints for all encrypted vector database operations using standard RESTful principles and JSON responses.

**API Version:** `v0.11.0`  
**Base Path:** `/v1`

## Base URL

**Development:** `http://localhost:8000/v1`  
**Production:** `https://your-domain.com:8000/v1`

## Authentication

All endpoints (except `/v1/health`) require an API key header:

```http
X-API-Key: cyborg_your_api_key_here
```

## Response Format

**Success:**
```json
{
  "status": "success",
  "message": "Operation completed successfully"
}
```

**Error:**
```json
{
  "status_code": 400,
  "detail": "Error description"
}
```

## Interactive Documentation

Explore the complete API at: [http://localhost:8000/docs](http://localhost:8000/docs)
