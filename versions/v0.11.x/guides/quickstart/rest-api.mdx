The **CyborgDB** REST API provides a simple HTTP interface for interacting with encrypted indexes. It allows you to add data, query vectors, manage metadata, and handle index operations from any language or platform.

## Overview
The REST API is exposed by the Docker service and can be used in any environment that supports HTTP requests. It is a flexible option for integrating CyborgDB into custom stacks, serverless workflows, or non-Python/JavaScript systems.
