Access metadata and configuration information about your encrypted index. These read-only properties provide details about the index without modifying it.

## index_name

```python
index.index_name
```

### Returns

`str` - The name of the index.

## index_type

```python
index.index_type
```

### Returns

`str` - The index type, or "unknown" if unavailable.

## index_config

```python
index.index_config
```

### Returns

`Dict[str, Any]` - Dictionary containing index configuration, or empty dict if unavailable.
