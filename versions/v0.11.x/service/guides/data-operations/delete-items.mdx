You can delete items from an encrypted index using `delete()`:

<CodeGroup>
```python Python
index.delete(["item1", "item2"])
```

```cpp C++
index.Delete({"item1", "item2"});
```
</CodeGroup>

<Warning>This operation is irreversible. Once you delete an item, you cannot recover it.</Warning>
