Returns a list of all document IDs stored in the vector store.

<Tabs>
    <Tab title="Embedded">

        ```python
        list_ids() -> List[str]
        ```

        ### Returns

        `List[str]`: List of all document IDs in the store

        ### Example Usage

        ```python
        ids = store.list_ids()
        print(f"Store contains {len(ids)} documents")
        print(f"First 5 IDs: {ids[:5]}")
        ```
